Registro manual

Este formulario ahora guarda vehículos reales en PostgreSQL, enlazando organización, propietario, telemetría y las cámaras que viajarán montadas en la unidad.

Cámaras asociadas

Selecciona las cámaras móviles que quedarán montadas en este vehículo. Si una cámara ya estaba en otra unidad, se reasignará al guardar.

Guía rápida

Al registrar un vehículo puedes dejarlo como ficha operativa o enlazarlo a una fuente de telemetría por API.

Dron

Úsalo para aeronaves no tripuladas y equipos de vuelo. Puedes consumir su posición desde una API externa que entregue id, latitud y longitud.

Automóvil

Úsalo para carros, camionetas o patrullas. Puedes dejarlo como ficha manual o conectarlo a una API GPS para verlo en el mapa.
